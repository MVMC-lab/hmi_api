remo_set
=========

呼叫介面
^^^^^^^^
:code:`[error] = remo_set(port, timeout)`

簡介
^^^^
    為避免串列埠斷線，讓輪詢成為永久迴路而當機。呼叫本函式，
    設定指定串列埠port之逾時時間timeout，當呼叫其他通訊函示，
    串列埠在設定時間內無回應時即退出，並回傳錯誤代碼。

輸入參數
^^^^^^^^
    **port**:愈進行設定之串列埠。
    
    **timeout**:逾時時間，單位為毫秒，預設逾時時間為1秒，可接受範圍為0毫秒~10秒。

回傳變數
^^^^^^^^
    **error**:錯誤代碼。

        0:設定成功。

        1:超過可接受範圍。
